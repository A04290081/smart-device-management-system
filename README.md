##smart device management system

The primary goal of this mini-project is to develop a Python application that simulates the 
management of different smart devices. The system includes a base class 
specialized child classes for 
TemperatureSensor , 
SecurityCamera , and 
SmartDevice and 
SmartLight . Each 
device has common attributes like device ID, name, and power status, while also 
possessing unique functionalities. The project emphasizes data encapsulation, proper use 
of inheritance with 
super() , and a user-friendly command-line interface for interaction.# smart-device-management-system
